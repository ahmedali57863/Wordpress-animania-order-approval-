# Animania Order Approval System (WordPress + WooCommerce) This project is a **WordPress + WooCommerce based eCommerce solution** that includes a **custom plugin** for managing order approvals. It was developed as part of a university assignment to demonstrate skills in WordPress customization, plugin development, and database integration. ## ✨ Features - 🛒 Full eCommerce store setup with WooCommerce - 🎨 Custom WordPress theme setup and styling - 📦 Product catalog with categories, tags, and checkout - ⚡ Custom WordPress plugin: **Animania Order Approval** - Admin dashboard for pending, approved, and rejected orders - Database table for order tracking (customer, product, total, paid, remaining) - Approve/Reject actions with email notifications - WPForms integration for capturing orders - 🔐 Secure (no sensitive credentials committed) ## 🛠️ Tech Stack - WordPress - WooCommerce - PHP - MySQL - WPForms - HTML, CSS, JS ## 🚀 Setup 1. Install WordPress locally (XAMPP / WAMP) or on a live server 2. Clone this repository into your `htdocs` (XAMPP) or `www` (WAMP) folder 3. Create a database and configure `wp-config.php` (not included for security) 4. Import the provided SQL if available 5. Activate the custom plugin from the WordPress dashboard ## 📌 Notes - `wp-config.php` and `uploads/` are ignored for security and storage reasons. - Replace with your own database credentials before running locally. ---
